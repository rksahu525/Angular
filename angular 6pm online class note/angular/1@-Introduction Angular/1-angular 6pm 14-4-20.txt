Setup Environment for Angular
--------------------------------------------
1. Download and Install Node JS [ for NPM ]
        https://nodejs.org/en/download/

    C:\> node -v
    C:\> npm -v

2. Install TypeScript
 
    C:\> npm install -g typescript@latest
    C:\> tsc -v

3. Install Angular CLI  [Command Line Tool]

    C:\> npm install -g @angular/cli@latest
    C:\> ng version

Note: Packages related details you can get from the official "npmjs.org"

If you already have an older version of Angular installed in your PC:

1. Un-install existing angular CLI
        > npm uninstall  -g @angular/cli
2. Run the commands
        > npm cache clear
        > npm cache verfiy
